# AdOS

Meta Ads automation app — Airtable-driven campaign, ad set, and creative management via Meta Graph API.

## Stack
- Single-file PWA (HTML/CSS/JS)
- Airtable (appWyCQ6WbGO02RGR) — Campaign → Ad Set → Media tables
- Meta Graph API v18.0
- n8n (WF-ADOS-EXECUTOR)
- Netlify deploy

## Status
15% — blocked on Meta System User Token

## How to run
Drop `index.html` into Netlify. Set `META_TOKEN` and `AIRTABLE_API_KEY`.
